### Getting Started

1. Install python library
```cmd
pip install requests
pip install beautifulsoup4
pip install HanziConv
pip install lxml
```
2. Compile .java
```cmd
javac Server.java
javac Client.java
```

3. Run file
```cmd
java Server
java Client
```
